Este projeto mergulha nos históricos e passados emocionais dos Mugiwaras de One Piece. Com detalhes das perdas, traumas e sonhos iniciais de cada membro da tripulação, mostrando como essas experiências moldaram suas personalidades e criaram o vínculo inquebrável que os move. É uma análise da fundação do bando do Chapéu de Palha.
